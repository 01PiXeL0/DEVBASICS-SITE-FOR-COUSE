О курсе

В этом курсе вы научитесь создавать современные веб-страницы с помощью HTML и CSS. Вы освоите основы верстки, научитесь работать с текстом, изображениями, формами и другими элементами веб-страниц.

Чему вы научитесь:

  • Создавать структуру веб-страниц с помощью HTML
  • Стилизовать элементы с помощью CSS
  • Работать с современными технологиями верстки
  • Создавать адаптивный дизайн
