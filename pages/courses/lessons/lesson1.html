Введение в HTML

Что такое HTML?

HTML (HyperText Markup Language) - это язык разметки, который используется для создания веб-страниц. С его помощью мы определяем структуру и содержание веб-страницы.

Структура HTML документа

<!DOCTYPE html>
<html>
    <head>
        <title>Название страницы</title>
    </head>
    <body>
        <h1>Заголовок</h1>
        <p>Параграф текста</p>
    </body>
</html>
  • <!DOCTYPE html> - объявление типа документа
  • <html> - корневой элемент
  • <head> - метаданные страницы
  • <body> - содержимое страницы

Базовые теги

<h1> - <h6>

Заголовки разных уровней

<p>

Параграф текста

<div>
